# 🚀 Micro Story Generator from Technical Concepts

> Transforming complex technical concepts into creative micro-stories ✨

---

## 📌 Overview

**Micro Story Generator** is a React-based web application that converts technical topics into short, engaging, and easy-to-understand micro-stories.  

This project demonstrates creativity + technical implementation by blending storytelling with software development.

---

## 🎯 Problem It Solves

Understanding technical concepts can sometimes feel overwhelming.  
This application simplifies them by turning them into relatable mini stories, making learning more engaging and fun.

---

## 🛠️ Tech Stack

- ⚛️ React.js  
- 🟨 JavaScript (ES6+)  
- 🎨 CSS3  
- 🌐 HTML5  

---

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/thanmai09/Micro-Story-Generator-from-Technical-Concepts.git
```

Navigate into the folder:

```bash
cd Micro-Story-Generator-from-Technical-Concepts
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

App will run on:

```
http://localhost:3000
```
